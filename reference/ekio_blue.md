# EKIO Primary Blue Scale

Brand identity color scale from light to dark.

## Usage

``` r
ekio_blue
```

## Format

Named character vector with 10 shades (50-900)

## Examples

``` r
ekio_blue["700"]  # Primary brand blue
#>       700 
#> "#1E3A5F" 
ekio_blue["200"]  # Light blue for backgrounds
#>       200 
#> "#A8D0E8" 
```
