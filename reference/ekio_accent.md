# EKIO Named Accent Colors

Individual accent colors for quick access.

## Usage

``` r
ekio_accent
```

## Format

Named character vector with 8 accent colors

## Examples

``` r
ekio_accent["blue"]    # Primary blue
#>      blue 
#> "#1E3A5F" 
ekio_accent["orange"]  # Contrast accent
#>    orange 
#> "#DD6B20" 
```
