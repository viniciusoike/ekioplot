# EKIO Teal Scale

Secondary brand color for complementary use.

## Usage

``` r
ekio_teal
```

## Format

Named character vector with 10 shades (50-900)

## Examples

``` r
ekio_teal["700"]  # Strong teal
#>       700 
#> "#2C7A7B" 
ekio_teal["300"]  # Light teal accent
#>       300 
#> "#81E6D9" 
```
