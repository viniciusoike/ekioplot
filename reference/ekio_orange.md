# EKIO Orange Scale

Accent color for highlights, warnings, and contrast against blue.

## Usage

``` r
ekio_orange
```

## Format

Named character vector with 10 shades (50-900)

## Examples

``` r
ekio_orange["600"]  # Primary orange accent
#>       600 
#> "#DD6B20" 
ekio_orange["300"]  # Soft orange for fills
#>       300 
#> "#FBD38D" 
```
