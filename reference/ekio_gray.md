# EKIO Neutral Gray Scale

Neutral grays for text, backgrounds, borders, and grids.

## Usage

``` r
ekio_gray
```

## Format

Named character vector with 10 shades (50-900)

## Examples

``` r
ekio_gray["900"]  # Darkest text
#>       900 
#> "#1A202C" 
ekio_gray["300"]  # Grid lines
#>       300 
#> "#E2E8F0" 
```
